## One-Click Deploy

### Command Line (recommended)
```bash
git init && git add . && git commit -m "feat: TikTok Spark Studio prototype"
git branch -M main
git remote add origin https://github.com/<YOUR_USERNAME>/spark-studio-prototype.git
git push -u origin main
# GitHub Actions auto-deploys — check Actions tab for live URL
```

### GitHub UI
1. Create a new repo → upload all files → Settings → Pages → Source: `GitHub Actions`
